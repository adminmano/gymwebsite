# APEX Fitness Website

A modern, professional, multi-page gym website for **APEX Fitness** located in **Puthery**.  
Designed with a premium black, red, and white theme, focused on conversion and mobile usability.

## Live Project Overview

This project includes:

- Bold hero section with call-to-action
- Conversion-focused homepage sections
- Dedicated About, Services, Pricing, Gallery, and Contact pages
- WhatsApp-integrated forms (Join + Contact)
- Sticky WhatsApp chat button on all pages
- Smooth scrolling and simple reveal animations
- Fully responsive layout for mobile, tablet, and desktop

## Pages Included

- `index.html` - Home
- `about.html` - About Us
- `services.html` - Services
- `pricing.html` - Membership Plans
- `gallery.html` - Gym Gallery
- `contact.html` - Contact + Google Maps

## Run Locally

1. Open the project folder.
2. Start a local server:
   - Python: `python -m http.server 5500`
3. Open in browser:
   - `http://localhost:5500`
